# DSA Notes

A collection of Data Structures & Algorithms problems, solutions, and notes — solved in **Python**.

## About

This repo contains my practice solutions to DSA problems (LeetCode, GFG, etc.), along with notes on:
- My approach/intuition for each problem
- Common mistakes made and how they were fixed
- Time and space complexity analysis
- Key takeaways/patterns to remember

## Structure

Currently organized loosely — each problem typically has its own markdown file containing the problem statement, approach, code, and notes. Folder structure will be refined over time as more problems are added.

```
dsa-notes/
├── README.md
├── check_duplicate.md
├── ...
```

## Format of Each Note

Each problem note generally follows this structure:

1. **Problem** — short description of the question
2. **Approach** — explanation of the logic/strategy used
3. **Code** — the Python solution
4. **Complexity** — time and space complexity
5. **Mistakes Made & Fixes** — errors encountered and corrections
6. **Key Takeaways** — patterns or concepts to remember for future problems

## Goal

The main goal of this repo is to:
- Track progress through DSA practice
- Reinforce learning by documenting mistakes and fixes
- Build a personal reference for recurring patterns and techniques

## Language

All solutions are written in **Python 3**.
