# DSA Patterns

A reference guide for recognizing common problem patterns based on trigger words/phrases, along with the typical tool and core idea used to solve them.

This file will keep growing as more patterns are identified during practice.

---

## Frequency Map Pattern

**Trigger words:**
- Anagram
- Frequency
- Count occurrences
- Most frequent
- Character count
- Word count

**Tool:** `dict` (or `collections.Counter`)

**Idea:** Count how many times each element appears.
